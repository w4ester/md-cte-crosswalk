# Maryland CTE Postsecondary Crosswalk

Static visualization of Maryland Career and Technical Education (CTE) postsecondary pathways.

**Live site:** https://w4ester.github.io/md-cte-crosswalk/

## Data Source

All pathway data comes from the official MSDE Postsecondary Crosswalk v3.01 (January 2026).

## Features

- 14 Career Clusters
- 48 Programs of Study
- Community College pathways
- University transfer options
- Industry-Recognized Credentials (IRCs)
- Excel export
- PDF print support
- Dark mode

## Development

```bash
cd frontend
npm install
npm run dev
```
